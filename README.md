# Bitcoin Payments Alliance Website

Static website for [bitcoinpaymentsalliance.org](https://bitcoinpaymentsalliance.org).

## Languages

- English (default, `/`)
- Deutsch (`/de/`)
- Français (`/fr/`)
- Italiano (`/it/`)
- Español (`/es/`)

Auto-detects browser language on first visit.

## Tech Stack

Pure HTML + CSS. No frameworks, no build step.
